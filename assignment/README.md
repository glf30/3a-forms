# Forms Practice

This assignment focuses on building form components in React using `useState`.

You will complete **two independent exercises**.

---

## Part 1: Message Sender Form

Build a simple form that captures and submits a single message.

### Requirements

* Create a `MessageForm` component

* Use `useState` to manage the input value

* The form must include:

  * A text input for **message**
  * A submit button

* The input value must be managed with React state

* On submit - display the message below the form.

* After submission, clear the input

* DO NOT allow the user to submit an empty message! If they do, display an alert

---

## Part 2: Profile Setup Form

Build a profile setup form that allows a user to enter their basic information.

### Requirements

* Create a `ProfileForm` component

* Use `useState` to manage form data

* The form must include:

  * An input for **name**
  * An input for age
  * An email input for **email**
  * A dropdown (`select`) for **role**
    * Example roles: `Student`, `Instructor`, `Guest`
  * A textarea for **bio**

* All input values must be managed with React state

* Store all form values in a single state object

* On form submission, display the input information below the form.
